# Autonomous-Airhockey
Senior Design project revolving around building autonomous airhockey robot to face other groups in tournament.

This project requires Visual Studio 2015, OpenCV libraries, and the CLEye SDK for Playstation Eye.

PS3EyeDemo_C.cpp: C++ file containing code for image processing, as well as angle and trajectory calculations.
CameraControl.c: Definitions for CLEye camera functions.
CameraControl.h: Header file linking to camera functions.
